Pour faire fonctionner le LDAP il faut mettre LDAPCompononent.php dans le
rep Controller/Component.

Ensuite dans AppController.php appeller le compononent et lui indiquer sur
quel LDAP se connecter, ex :

class AppController extends Controller {

	public $components = array('Session', 'Auth', 'LDAP' => array(
		'host' => 'ldap.42.fr',
		'port' => 389
		));
}

Et apres tu as dans LdapUsersController.php les quelques fonctions d'exemple
que j'avais fait pour le framework.
